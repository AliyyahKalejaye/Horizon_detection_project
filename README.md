# Horizon_detection_project
The Beginner’s Guide to Autonomous Horizon Detection and Stabilization for UAVs with OpenCV.
This project started from detection in images, to detection in videos, to actual detection with PC cam, external cam, and finally with Raspberry Pi 4 and Pi Cam.

You will find test images, videos, and their corresponding codes. Also, the training model code and the .pkl file.
You will also find updated code with filters and jitters implemented in the code. So, you can see how updates are made.
